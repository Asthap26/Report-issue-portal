📌 Local Issue Portal
A full-stack web application where citizens can report local issues and authorities manage and assign them to companies for resolution.

🚀 Features
Secure authentication (hash + salt)
Citizen complaint system
Government dashboard for management
Company dashboard for status updates

🛠️ Tech Stack
Backend: Flask (Python)
Frontend: HTML, CSS, JavaScript
Template Engine: Jinja2
Database: PostgreSQL (pgAdmin)
ORM: SQLAlchemy

📂 Project Goal
To build a real-world system connecting citizens, government, and companies for efficient issue resolution.

⚙️ Setup Instructions
# Clone repo
git clone <your-repo-link>

# Create virtual environment
python -m venv venv
source venv/bin/activate   # (Linux/Mac)
venv\Scripts\activate      # (Windows)

# Install dependencies
pip install -r requirements.txt

# Run app
python app.py
