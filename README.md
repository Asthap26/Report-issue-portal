📌 Local Issue Portal
A full-stack web application that allows citizens to report local public issues and enables government authorities to manage and resolve them efficiently by assigning tasks to companies.
🚀 Features
👤 Authentication System
Secure signup & login for all users
Passwords stored using hashing + salt
Session-based authentication (Flask)
🧑‍🤝‍🧑 Citizen Module
Register with mobile OTP (simulated)
Submit complaints (roads, lights, water, etc.)
View and track submitted complaints
🏛️ Government Dashboard
View all complaints
Identify most reported issues
Assign complaints to companies
🏢 Company Dashboard
View assigned tasks
Update complaint status:
Pending
In Progress
Solved
🛠️ Tech Stack
Backend: Python, Flask
Frontend: HTML, CSS, JavaScript
Template Engine: Jinja2
Database: PostgreSQL (pgAdmin)
ORM: SQLAlchemy
Security: Password Hashing with Salt
🧠 Key Concepts Used
Role-Based Access Control
Database Relationships (Foreign Keys)
ORM (SQLAlchemy)
Session Management
Data Structures (lists, dictionaries for analytics)
📂 Project Objective
To build a real-world complaint management system that improves communication between citizens, government authorities, and service providers.
